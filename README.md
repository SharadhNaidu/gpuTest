# AI Desktop Assessment Suite

Comprehensive testing tool for evaluating AI/ML workload suitability on desktop systems.

## Quick Start

### Linux / macOS
```bash
git clone <repo-url>
cd gpuTest
python3 run.py
```

### Windows
```cmd
git clone <repo-url>
cd gpuTest
python run.py
```

Or double-click `run.bat`

## Output

Reports are saved to:
- `ai_desktop_report.png` - Visual summary
- `ai_desktop_report.txt` - Text report
- `ai_desktop_report.json` - Machine-readable data
